\Logger
jump to top

Package: GoogleApiAdsCommon\Util
A logger class which registers appenders (any writable handle location which can be fopen'ed) with a unique name for that log.

The library will write all incoming/outgoing SOAP XML to SOAP_XML_LOG and all records of requests to REQUEST_INFO_LOG.

Currently, levels are only used as indicators (there is no filtering mechanism as of yet).

A list of supported appenders (file handles) can be found here: http://us2.php.net/manual/en/wrappers.php
