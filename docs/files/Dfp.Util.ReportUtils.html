DownloadReport(string $downloadUrl, string $filePath = NULL) : mixed
static

Downloads a report from a URL.

If the filePath parameter is specified it will be downloaded to the file at that path, otherwise it will be downloaded to memory and be returned as a string.

Parameters
NameTypeDescription
$downloadUrlstring

the URL of the report download

$filePathstring

an optional path of a file to download the report to

Returns
TypeDescription
mixedif $filePath isn't specified it will return the contents of the report, otherwise the size in bytes of the downloaded report
__construct() : void

The ReportUtils class is not meant to have any instances.
