Class FlashExpandableCreative

Description
Description | Vars (details) | Methods (details)

An expandable {@code Creative} where the collapsed size is a Flash SWF file and the expanded size is another Flash SWF file. If Flash isn't supported in the browser, the fallback image will be used.

This creative will not work with iframe ad tags.

This creative is only available to small business networks.
