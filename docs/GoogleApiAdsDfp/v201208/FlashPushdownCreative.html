Class FlashPushdownCreative

Description
Description | Vars (details) | Methods (details)

{@code Creative} that pushes page content down when the creative expands.

The collapsed size is one Flash SWF file and the expanded size is another Flash SWF file. <p> This creative will not work with iframe ad tags. <p> This creative is only available to small business networks.
