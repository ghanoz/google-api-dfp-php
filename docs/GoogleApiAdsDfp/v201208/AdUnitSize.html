Class AdUnitSize

Description
Description | Vars (details) | Methods (details)

An {@code AdUnitSize} represents the size of an ad in an ad unit. Starting with v201108 this also represents the environment, and companions of a particular ad in an ad unit. In most cases, it is a simple size with just a width and a height (sometimes representing an aspect ratio).
