					HasDestinationUrlCreative
					
											A {@code Creative} that has a destination url
										
				

								

					InternalRedirectCreative
					
											A {@code Creative} hosted by either DoubleClick for Advertisers (DFA) or DART for Publishers.
										
				

								

					LegacyDfpCreative
					
											A {@code Creative} that isn't supported by Google DFP, but was migrated  from DART. Creatives of this type cannot be created or modified.
										
				

								

					TemplateCreative
					
											A {@code Creative} that is created by the specified creative template.
										
				

								

					ThirdPartyCreative
					
											A {@code Creative} that is served by a 3rd-party vendor.
										
				

								

					VastRedirectCreative
					
											A {@code Creative} that points to an externally hosted VAST ad and is  served via VAST 2.0 XML as a VAST Wrapper.  This creative is read only.
										
				

								

					BaseDynamicAllocationCreative
					
											A base class for dynamic allocation creatives.
										
				

								

					BaseRichMediaStudioCreative
					
											A {@code Creative} that is created by a Rich Media Studio.
										
				

								

					ClickTrackingCreative
					
											A creative that is used for tracking clicks on ads that are served directly  from the customers' web servers or media servers.
